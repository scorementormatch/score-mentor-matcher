This tool will match the best possible mentor to a client based on what the client wants to do. 
To update the Excel file (Chapter 24 Skill Sets.xlsx), you just need to upload the new version to your GitHub repository. Streamlit is connected directly to your GitHub repository, so whenever you update a file on GitHub, your live app automatically updates!
Here is the easiest way to replace the Excel file:
Step 1: Replace the File on GitHub
Go to your GitHub repository page:
[https://github.com/scorementormatch/score-mentor-matcher](https://github.com/scorementormatch/score-mentor-matcher)
Click the Add file button near the top right of the file list and choose Upload files.
Drag and drop your newly updated Chapter 24 Skill Sets.xlsx into the box (make sure it has the exact same filename as the original).
At the bottom of the page, click the green Commit changes button.
Step 2: What Happens Next?
Automatic Sync: Streamlit will automatically detect the new commit on GitHub and update your live app within about 1 to 2 minutes.
Force Refresh (if needed): If you don't see the updated data right away in your app:
Open your live Streamlit app.
Click the three dots (⋮) in the top-right corner of the app screen.
Select Clear cache (or Rerun), then refresh your browser tab.
That's all there is to it! Any future updates to the Excel sheet can be done using those same steps.
